Skalper-Unlocker — это специализированный алгоритмический комплекс для безопасного вывода счета из просадки и разруливания хедж-«замков».
Главная идея:
Вместо пассивного ожидания отката рынка, советник активно «стягивает» среднюю цену убыточных ордеров к текущей котировке. Это достигается за счет их постепенного, математически выверенного частичного закрытия.
Как это работает:
Генерация ресурса: Встроенные скальпер-ядра (Core 1 и Core 2) генерируют пул прибыли на волатильных движениях.
Эффективное откусывание: Средства из пула направляются на закрытие самых дальних убыточных позиций. Каждая операция совершается только при условии сохранения чистого плюса для баланса.
Жесткий контроль рисков: Многоуровневая система защиты и динамический ребаланс не допускают разбалансировки Buy/Sell выше заданных пользователем лимитов, исключая разгон просадки.
Итог: Плавное снижение своповой и ценовой нагрузки на депозит, гарантированное восстановление баланса до стартового уровня и автоматический переход в штатный режим торговли.

## Инструкция по установке
1. Скачай последний релиз файла `Skalper-Unlocker.ex4` из раздела **Releases**.
2. Скопируй его в папку `MQL4/Experts` твоего терминала MetaTrader 4.
3. Перезапусти терминал.
